Import Data Manually

At times, it will be necessary to manually import data. To do so, the file must be saved locally.

Import Data Manually

  1. Select data mode: Trial Balance, Budgets, or Occupancy
  2. Navigate to Sheet Management > Import
  3. Select management company name from drop-down list
  4. Select parsing configuration from drop-down list
  5. Select year and month of oldest file to be imported
  6. Click Browse... and select source file location
  7. Click +Add File to create additional entries, if needed
  8. Click Upload
  9. Navigate to Home > Tracking and verify successful import
