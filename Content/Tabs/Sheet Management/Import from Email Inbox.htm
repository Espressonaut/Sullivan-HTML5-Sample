Import Data from Email Inbox

Data will typically be imported from the BC Reporting inbox. Attachments are only displayed for import when viewed with the correct data mode. If the attachment is not recognized by the system, it will be labeled as "Uncategorized," and will be displayed in all data modes.

Import Data

  1. Select data mode: Trial Balance, Budgets, or Occupancy
  2. Navigate to Sheet Management > Import from Email Inbox
  3. If displayed file needs to be replaced, click Choose File and select correct version
  4. Select Year and Month for each file to be imported
  5. Verify management company in drop-down list
    • If not automatically selected, review management company from email and assign a contact
  6. Select Property
  7. Select Parsing Configuration
  8. Click checkbox for file(s) to be imported
  9. Click Import
  10. Navigate to Home > Tracking and verify successful import

If a file is Uncategorized, or is appearing under the wrong data mode, it must be assigned the correct data type.

Assign Data Type

  1. Select appropriate data type (or Uncategorized) from "Category" drop-down list
  2. File will automatically update and only be displayed when importing from newly-assigned data mode

It is also possible to save files to a local computer at this time. This is sometimes necessary if changes must be made to the original file before uploading.

Save Files Locally

  1. Click checkboxes for needed files
  2. Click Download Selected
  3. Choose destination for saved files
