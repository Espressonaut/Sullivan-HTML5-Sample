Assign Source Properties

The data collection software identifies property names from email file attachments sent by management companies. It uses these Source Property names to match them to the correct Client Property names. Source Property names must first be assigned manually to their matching Client Property names.

Assign or Reassign Source Properties

  1. Navigate to Company Info > Assign Source Properties
  2. Select management company (or Unassigned) from Management Company drop-down list
  3. Locate Source Property and select matching property in Client Property drop-down list
  4. Click Save

There will be a delay of 30 minutes for changes to take effect.
