Preparing for Data Input

Before data can be collected and reported, the data collection software must be set up to accept and process the information. A management company must be created, properties must be created and assigned, and parsing instructions must be configured.

Click on any of the steps below for specific instructions, or follow the bookmarks on the left of the screen for more detailed information.

The steps to prepare the software to receive data are:

  1. Create a Management Company
  2. Create Contacts for the Management Company
  3. Create a Property
  4. Assign Companies
  5. Assign Contacts for the Property
  6. Map Accounts for Management Company
  7. Create Parsing Rules for Property
