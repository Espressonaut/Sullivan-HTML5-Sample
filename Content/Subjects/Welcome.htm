Lexington Solutions Data Collection

The Lexington Solutions Data Collection System reads raw trial balance and budget files in a number of formats, and maps data directly to multiple target accounts. It also allows for manual importation of occupancy reports to be viewable in Fusion. The system performs a number of data checks, and manages different accounting rules for various properties to ensure data is correct.

The table of contents on the left of this help guide will walk you through the two main functions of the data collection software: preparing the system to receive data, and importing data into the system. These bookmarks contain additional information, including a "Tab Reference" which will allow you to see the commands of each individual tab in the software.

You can select a particular topic by clicking on the appropriate bookmark, or may navigate through topics sequentially by using the arrows in the upper right of the screen.

There are an index and glossary to assist you, and the search box on the upper right of the screen is available at all times.
