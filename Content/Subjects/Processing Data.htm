Processing Data

The Data Collection System pulls information from spreadsheets, and is typically imported from emails to the BC Reporting inbox. In rare situations, it my be necessary to manually import the data with a spreadsheet that has been saved locally. If corrections are made, the data must also be reprocessed.

Click on any of the steps below for specific instructions, or follow the bookmarks on the left of the screen for more detailed information.

The ways to process data are:

  1. Import Data From Email Inbox
  2. Import Data Manually
  3. Reprocess Data

Once data has been sent for processing, it is important to verify the data has been processed correctly.

Note: the data collection software performs its mapping process every 30 minutes, and exports data to Fusion every two hours. These processes can also be manually triggered from the Admin tab's Control Panel.
